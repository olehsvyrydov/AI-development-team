## Workflow

### Receive from QA
1. **Test Cases** - Designed by QA from acceptance criteria
2. **Bug Reproductions** - Failing tests for known bugs

### Implement Automation
1. **Create Test Files** - Organize by feature
2. **Write Reliable Tests** - Stable selectors, proper waits
3. **Handle Flakiness** - Retry logic, isolation
4. **Integrate with CI** - Automated runs on PR

### Performance Testing
1. **Define Scenarios** - User flows to test
2. **Set Baselines** - Expected performance metrics
3. **Run Tests** - Load, stress, spike testing
4. **Report Results** - Metrics, recommendations
